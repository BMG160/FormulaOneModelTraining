# Formula One Model Training

This project contains the machine learning notebooks and source code for Formula One ranking prediction.

Large dataset and trained model files are not uploaded to GitHub due to file size limits. They can be generated by running the notebooks in order:

1. data_preprocessing.ipynb
2. xgboost_model.ipynb
3. catboost_model.ipynb
4. gradientboosting_model.ipynb
5. lightbgm_model.ipynb
6. randomforest_model.ipynb
7. xgboostranker_model.ipynb ("!uvicorn main:app --reload --host 0.0.0.0 --port 8000" must be run before running Flutter application")

The trained model files are generated locally after model training.
